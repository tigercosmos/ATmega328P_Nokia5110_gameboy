# ATmega328P Nokia5110 Gameboy
DIY Handheld Game Console Made With ATmega328P and Nokia5110 LCD

## Enviroment

Sofeware: 
- Windows 10
- AtmelStudio 7
- AVRDUDESS 2.4 (avrdude version 6.1)

Hardware:
- ATmega328P
- Arduino UNO (ISP)
- Nokia5110 LCD
- BJT
- Buttons
- Buzzer
- Resistance
- Wires
- 220u Capacitance
